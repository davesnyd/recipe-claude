  5. Authentication Options

  When you push, GitHub will ask for authentication. You have two options:

  Option A: Personal Access Token (Recommended)

  1. Go to GitHub Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Generate new token with repo permissions
  3. Use your GitHub username and the token as password when prompted

  Option B: SSH Key (More Secure)

  1. Generate SSH key: ssh-keygen -t ed25519 -C "your-email@example.com"
  2. Add to SSH agent: ssh-add ~/.ssh/id_ed25519
  3. Copy public key: cat ~/.ssh/id_ed25519.pub
  4. Add to GitHub Settings → SSH and GPG keys
  5. Use SSH URL instead: git remote set-url origin git@github.com:YOUR_USERNAME/YOUR_REPO_NAME.git
